# Form Module Field: Image picker

UI-kit layout for form elements. 

## Usage

Import module

```typescript
import { NgModule } from '@angular/core';
import { FormComponentsImagePickerModule } from '@pe/ng-kit/modules/form-components/image-picker';

@NgModule({
  imports: [FormComponentsImagePickerModule]
})
export class YourModule {
}
```

Available components:

```html
<pe-image-picker></pe-image-picker>
```
