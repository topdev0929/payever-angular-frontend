## Usage

```typescript

import { FormComponentsTooltipIconModule } from '@pe/ng-kit/modules/form-components/tooltip-icon';

@NgModule({
  imports: [FormComponentsTooltipIconModule]
})
export class YourModule {
}
```

Available components:

