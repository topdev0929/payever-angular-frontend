AngularJS modals

The main idea is to create route for modal and wrap page content by {{ exampleTag }}. After that regular page become to look like a modal. Full documentation is here.

To make modal always appear over the page content, use children to declare page with modal component in routes. Also in following example ListComponent must have {{ routerOutletTag }} in template.
To make one modal appear over another modal you should put {{ routerOutletTag }} in the end of bottom modal template but out of {{ exampleTag }}.
